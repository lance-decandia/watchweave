# WatchWeave
Anime tracking platform built with DevSecOps practices.

## Team
- Lance DeCandia
- Ioannis Magkaniotis
- Jackson Tullman

## Stack
- Frontend: React + TypeScript
- Backend: Node.js + Express
- Recommendation Engine: Python + Flask
- Database: PostgreSQL
- Cache: Redis
- CI/CD: GitHub Actions + GitLab CI/CD
- Infrastructure: AWS ECS, ECR, RDS, ElastiCache
- Monitoring: Prometheus + Grafana
# git-secrets
This repo uses git-secrets to prevent committing AWS credentials and other secrets.
